# Custom YouTube Live Stream Plugin

Custom YouTube Live Stream Plugin is a WordPress plugin that allows you to embed YouTube Live streams on your WordPress website.

## Installation

1. Download the plugin zip file from the [releases page](https://github.com/rizennews/custom-youtube-live-stream-plugin/releases).
2. Log in to your WordPress admin panel.
3. Go to Plugins > Add New.
4. Click on the "Upload Plugin" button and select the downloaded zip file.
5. Click "Install Now" and then "Activate" to activate the plugin.

## Usage

Once the plugin is activated, you can embed YouTube Live streams on your WordPress pages or posts by using the provided shortcode `[youtube_live_stream]`.

### Example Usage:


The above shortcode will embed the latest YouTube Live stream from your configured channel.

## Configuration

Before using the plugin, you need to configure your YouTube API Key and Channel ID in the plugin settings. You can obtain an API key from the Google Developer Console and find your Channel ID in the URL of your YouTube channel.
